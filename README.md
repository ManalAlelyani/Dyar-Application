# DyarApplication

DyarApplication is a fully-featured iOS application designed and developed from scratch using Swift and Xcode. It provides a smooth user experience with an intuitive and visually appealing interface.

## Features
- Multiple interactive screens with seamless navigation
- Clean and modern UI/UX design
- Compatible with iPhone and iPad devices
- Well-structured Swift code following best practices

## Technologies Used
- Swift
- Xcode
- Storyboard / SwiftUI
- Git & GitHub for version control

## Getting Started
1. Clone or download the repository
2. Open DyarApplication.xcodeproj in Xcode
3. Select a simulator or a physical iPhone device
4. Click the Run button (▶️) to build and launch the app
